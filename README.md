# SOCKET-in-Python

## Serial 통신
  - VSPE를 통한 가상 통신포트 설정, SerialPortMon을 통한 가상 클라이언트 설정
  - NMEA를 통한 가상 데이터 Serial통신으로 Server로 전달
  - TCP/IP 통신으로 SerialPortMon에 데이터 전달
  
  ## TCP/IP 통신
  - VSPE를 통한 가상 통신포트 설정, SerialPortMon을 통한 가상 클라이언트 설정
  - NMEA를 통한 가상 데이터 TCP통신으로 Server로 전달
  - TCP/IP 통신으로 SerialPortMon에 데이터 전달
